# AutoApplyJob.naukri.com
This is a Semi-Agentic AI Project That Helps you to automate daily task for applying jobs on Naukri Portal

1. Do Update Your ID password and Other Details in .env file.
2. Update the browser Path according to your Machine in main file i.e apply-Naukri_jobs.py.
3. Do install all neccesary imports.
4. in .json file update Q/A for autofillups if it doesn't able to autofill on naukri it skips that job post and send link directly to your telegram bot so you can check and apply on that job too.
5. you will see the direct links to all jobs on which the bot sent your applications in visited_jobs.txt this .txt file refresh after 7 days or every monday.
